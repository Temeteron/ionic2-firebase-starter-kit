Ionic and Firebase

Welcome to Ionic 2/3 and Firebase starter kit

In this starter-kit we are using as frontend framework, the ionic 2/3 with angularJs 4. As a database we use the famous firebase of Google. We are trying to demonstrate the usefulness of this platform.

You can add users from the add button on the top right corner. The list of the users is saved at firebase. Even in development mode, the list will load and refresh automatically the users after getting them from firebase. You can edit a user by clicking on him.
